Element: Box Model

This box demonstrates the CSS Box Model with margin, border, padding, and content.

The CSS Box Model consists of:

  • Content - The actual content of the box
  • Padding - Space around the content
  • Border - A border that goes around the padding
  • Margin - Space outside the border

Element: Float

This text demonstrates the float property. The image is floated to the left, and the text wraps around it. The float property is commonly used for text wrapping around images, creating multi-column layouts, and other design patterns. This is an example of using the float property in a practical scenario.

Here's another example with the image floated to the right. The text now wraps around the image on the left side. When using float, it's important to properly clear the float after using it to prevent layout issues. The clearfix technique is commonly used to solve this problem.
